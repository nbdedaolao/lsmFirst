# DEMO实现3个功能  
1.页面初始化时加载存放在指定的路径下的单词以及注释的文档   
2.注释点击覆盖   
3.添加单词   
  
单词记录.bat结构为：  
start python解释器路径 cem.py路径 word.txt路径   
-----------------------------------------------------------
# 强调  
**1.  word的文件存储格式建议为TXT,资源加载时按行加载，如果该行不符合格式将不会加载**  
**2.  添加单词时的格式建议为：小写单词,中文  format：[a-z]+,[\u4e00-\u9fa5]+**
